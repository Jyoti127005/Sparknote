# Sparknote
Building this was a deep dive into:
• Structured UI: Crafting a clean, distraction-free interface using HTML & CSS.
• Core Logic: Implementing the backend functionality and data management using Java.
• User Experience: Focusing on how we capture thoughts instantly without the "clutter" of traditional apps.

It is a note taking app , which is little diffrent from normal note taking app . Firstly it welcoming you in the app . Before writing the notes there is a tab of note title then start writing note . There will be so many option on toolbar like you can change background colour ( page colour ) , text colour , undo, redo , text size . You can save your note using save option . There is the advance option of ‘read’ that you want to summarise your notes then you simply select your note and click read option , it will read for you.
